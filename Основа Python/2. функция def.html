🏭 Фабрика Функций (DEF) 🏭

Узнай, как программисты прячут тысячи строк кода в одну маленькую команду!

Начальные станки на фабрике

Функция начинается со слова def (define - определить). Это как кнопка, которая запускает сложный механизм.

🎆

1. Скрытый механизм

Ничего не принимает, ничего не возвращает. Просто делает огромное действие внутри.

def start_fireworks():
    print("Запускаем 100 ракет!")
    print("Бабах!!!")
🎨

2. Входная (Принимает)

Мы даем ей параметр (материал), и она делает с ним действие, но обратно ничего не отдает.

def paint_car(color):
    print(f"Машина покрашена в {color}!")

Продвинутые станки (Return)

Самые полезные функции — те, которые обрабатывают данные и возвращают (return) результат нам в руки!

⚙️

3. Входно-Выходная

Берет число, делает математику и выплевывает результат обратно с помощью return.

def double_it(number):
    result = number * 2
    return result
☕

4. По умолчанию (Default)

Если мы забудем сказать функции, сколько нужно сахара, она возьмет значение по умолчанию!

def make_coffee(sugar=0):
    return f"Кофе готов. Сахар: {sugar}"

Собери функцию сам! 👨‍💻

Напиши Входно-Выходную функцию, которая принимает имя робота, собирает его и возвращает (return) готовую строку.

Блоки кода

    return my_robot
def build_robot(name):
    my_robot = "Робот " + name + " собран!"

Редактор Python

1. Создай функцию (def)
2. Логика (собираем строку)
3. Возвращаем результат (return)
