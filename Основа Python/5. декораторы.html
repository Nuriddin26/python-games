🪄 Магия Наклеек (Декораторы) 🪄

Пойми, почему разработчики обожают декораторы и ненавидят переписывать код!

Проблема: Босс требует добавить Джетпаки! 😠

У тебя есть 3 разных робота. Босс сказал: "Перед каждым действием робот должен включать джетпак!".
Без декоратора тебе придется вскрывать код каждого робота и переписывать его. Попробуй сам:

🚶
def move():
    print("Иду пешком")
Уф... пришлось лезть в логику!
🧹
def clean():
    print("Убираю мусор")
А если я случайно удалю нужную строку?
📦
def carry():
    print("Несу коробку")
Как же это долго и скучно!

А представь, если таких роботов 1000?! Переписывать вручную — это кошмар!

Решение: Волшебная наклейка (Декоратор) 🌟

Программисты придумали Декораторы. Это "супер-броня", которую написали один раз. Чтобы надеть её на робота, не нужно ломать его код! Просто прилепи сверху наклейку @.

Перетащи эту наклейку на роботов внизу:

@jetpack
Бросить @ сюда
🚶
def move():
    print("Иду пешком")
Бросить @ сюда
🧹
def clean():
    print("Убираю мусор")
Бросить @ сюда
📦
def carry():
    print("Несу коробку")

✅ ГЕНИАЛЬНО! Мы прокачали роботов, вообще не трогая их внутренний код!

Главный секрет FastAPI 🌐

Ваша функция `get_cat()` просто возвращает текст. Она не умеет работать с интернетом. Но взрослые разработчики написали гигантский декоратор-сервер. Перетащи наклейку @app.get на код, чтобы увидеть магию!

@app.get("/cat")
Бросить @app.get сюда
def get_cat():
    return "Вот твой котик!"
🌐 http://127.0.0.1:8000/cat
"Вот твой котик!" 🐈
