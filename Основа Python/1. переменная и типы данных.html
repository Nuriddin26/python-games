📦 Склад Переменных 📦

Узнай, как компьютер хранит информацию!

Переменная — это коробка! 📦

Чтобы компьютер что-то запомнил (например, очки в игре или имя игрока), ему нужна переменная. У любой переменной есть 3 главных свойства:

  • 🏷️ ИМЯ: Наклейка на коробке, чтобы мы могли её найти (например, score).
  • 🍎 ЗНАЧЕНИЕ: То, что лежит внутри коробки (например, число 100).
  • 📏 РАЗМЕР: Сколько места коробка занимает на складе (в памяти компьютера).

Создай свою переменную:



имя_переменной
❓

Попробуй изменить имя и значение, и посмотри, как меняется коробка!

Сортировка: Типы данных 🗂️

Мы не можем хранить суп в бумажном пакете! Компьютер тоже требует правильные коробки для разных данных.

Распредели значения по правильным коробкам (типам данных):

100
"Привет"
3.14
True
-5
"Кот"

str (Строка)

Слова и текст в кавычках

int (Целое число)

Числа без точки

float (Дробное)

Числа с точкой

bool (Логика)

True (Правда) / False (Ложь)

Третье свойство: Размер и Память 📏

У компьютера ограниченный склад (Оперативная память / ОЗУ). Каждая переменная занимает место.

Другие языки (C++, Java)

Программист сам выбирает размер жесткой коробки. Если положить слишком большое число, коробка порвется (Переполнение)!
