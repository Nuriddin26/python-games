🏗️ Завод Чертежей (Классы) 🏗️

Узнай, как создавать свои собственные типы данных и охранять их!

Класс — это чертеж. Объект — это готовая деталь.

Представь инструкцию по сборке LEGO-робота. Сама инструкция (чертеж) играть не может, она только из бумаги. Но по этой инструкции мы можем собрать 100 разных роботов (объектов)!

В Python Класс (class) — это такой чертеж.

Чертеж (Class Robot)

class Robot:
  name: str
  color: str

Конвейер готовых объектов

Зачем нужен Pydantic? 🕵️‍♂️

В обычном Python мы можем сказать, что возраст кота — это слово "мяу". Python промолчит, а потом программа сломается! Библиотека Pydantic (BaseModel) работает как строгий инспектор на таможне. Он проверяет, чтобы данные ТОЧНО совпадали с чертежом!

👮‍♂️

Таможня Pydantic: Проверка Кота!

Попробуй обмануть инспектора. Введи вместо возраста (int) какое-нибудь слово и нажми кнопку.



Собери Класс с защитой! 👨‍💻

Собери чертеж кота (Cat), который наследует защиту от инспектора (BaseModel). Перетаскивай блоки в правильном порядке!

Блоки кода

class Cat(BaseModel):
    age: int
from pydantic import BaseModel
    name: str

Редактор Python

1. Импорт строгого инспектора
2. Создаем Класс (Чертеж)
3. Свойство: имя (строка)
4. Свойство: возраст (число)
