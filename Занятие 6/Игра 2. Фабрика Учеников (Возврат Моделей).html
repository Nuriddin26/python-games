🎒 Фабрика Учеников (Pydantic) 🎒

Как Python превращается в JSON?

Нажми на объект Python, чтобы пропустить его через Pydantic-конвейер!

Pupil(
  name="Вася",
  grade=7
)

Нажми на меня!
➡️
{
  "name": "Вася",
  "grade": 7
}

FastAPI делает это автоматически, если ты используешь классы BaseModel!
