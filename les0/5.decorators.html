⚡ ЗАДАЧА: Добавь функцию джетпака каждому роботу вручную!

Представь, что тебе нужно обновить навыки роботов. Без декораторов придется залезать внутрь каждой функции, изменять её код и рисковать сделать опечатку[cite: 9].

🚶‍♂️
def move():
    print("Иду")
🧹
def clean():
    print("Убираю")
📦
def carry():
    print("Несу")

Ой! При ручном редактировании мы случайно сделали опечатку на третьем роботе (prnt вместо print), и программа сломалась. Изменять существующий код — это всегда риск!

⚡ ЗАДАЧА: Прокачай роботов с помощью Декоратора!

В отличие от ручного изменения, декоратор надевается поверх функции. Оригинальный код (def) остается абсолютно нетронутым, поэтому риск что-то сломать минимален!

Перетащи Декоратор ➡️
@jetpack
Бросить @ сюда
🚶‍♂️
def move():
    print("Иду")
Бросить @ сюда
🧹
def clean():
    print("Убираю")
Бросить @ сюда
📦
def carry():
    print("Несу")

✅ ОТЛИЧНО! Роботы получили новые способности, а их внутренний код остался чистым и безопасным!

⚡ ЗАДАЧА: Преврати функцию в настоящий Сервер!

Функция get_cat() просто возвращает текст. Она не умеет работать с интернетом. Перетащи полезный декоратор @app.get на неё, чтобы наделить её силой веб-сервера!

@app.get("/cat")
Бросить @ сюда
def get_cat():
    return "Вот твой котик!"
http://127.0.0.1:8000/cat
"Вот твой котик!" 🐈
